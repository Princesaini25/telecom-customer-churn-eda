# Telecom Customer Churn Analysis (EDA)

##  Project Overview

This project performs Exploratory Data Analysis (EDA) on the Telecom Customer Churn dataset to identify key factors that influence customer churn. The goal is to understand customer behavior and provide actionable insights to improve retention.

---

##  Dataset Information

* Dataset: Telecom Customer Churn
* Contains customer demographics, services, account details, and churn status

---

## Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Key Analysis Performed

* Data cleaning and preprocessing
* Handling missing values
* Converting data types
* Univariate and bivariate analysis
* Correlation analysis using heatmap
* Customer segmentation based on tenure and services

---

## Key Insights

* Customers with month-to-month contracts have higher churn rates
* New customers (1–2 months tenure) are more likely to churn
* Senior citizens have a slightly higher churn percentage
* Fiber optic users show higher churn compared to other internet services
* Lack of services like Online Security and Tech Support increases churn
* Customers using electronic check payment method have higher churn

---

## Recommendations

* Offer incentives to convert month-to-month customers into long-term contracts
* Improve onboarding experience for new customers
* Enhance service quality, especially for fiber optic users
* Promote value-added services like Online Security and Tech Support
* Encourage customers to switch to more reliable payment methods

---


## Conclusion

Customer churn is influenced by contract type, tenure, service quality, and additional services. By focusing on customer experience and strategic improvements, businesses can significantly reduce churn and improve retention.
